> Suppose that in the rod-cutting problem of Section 15.1, we also had limit $$l_i$$ on the number of pieces of length $$i$$ that we are allowed to produce, for $$i = 1, 2, \ldots, n$$. Show that the optimal-substructure property described in Section 15.1 no longer holds.

In the original rod-cutting problem, optimal substructure holds because the subproblems are independent. After we cut off a piece of length $$i$$ from a rod of length $$n$$, the optimal solution for the remaining rod of length $$n - i$$ is completely unaffected by our first choice. We can freely optimize the remainder using any combination of piece lengths.

When we introduce limits $$l_i$$ on the number of pieces of each length, this independence breaks down. The choice of how to cut the first piece affects what pieces remain available for cutting the rest of the rod. The subproblems are no longer independent because they share a constrained resource: the limits on piece counts.

Consider a concrete example. Suppose we have a rod of length 4, with prices $$p_1 = 2$$, $$p_2 = 5$$, $$p_3 = 6$$, $$p_4 = 7$$, and limits $$l_1 = 1$$, $$l_2 = 1$$, $$l_3 = 1$$, $$l_4 = 1$$ (we can produce at most one piece of each length).

Without limits, the optimal solution would be to cut the rod into two pieces of length 2, yielding revenue $$5 + 5 = 10$$. But with our limit $$l_2 = 1$$, we can only use one piece of length 2.

Now suppose we try to apply the original optimal-substructure argument. If we cut off a piece of length 2, we have revenue 5 and a remaining rod of length 2. The optimal solution for a rod of length 2 with these prices and limits would be to cut a piece of length 2, giving revenue 5. But we have already used our one allowed piece of length 2! The "optimal" solution for the remaining rod assumes a resource that we have already consumed.

If we proceed legally after using one length-2 piece, the best we can do with the remaining length-2 rod is to cut two pieces of length 1 (if $$l_1 \ge 2$$) or use whatever pieces remain available. This gives a lower revenue than the naive "optimal substructure" would suggest.

The original recurrence for rod cutting is:

$$r_n = \max_{1 \le i \le n}(p_i + r_{n-i})$$

This assumes that $$r_{n-i}$$ represents the true optimal revenue from a rod of length $$n - i$$, using any pieces. But with limits, $$r_{n-i}$$ depends on what pieces we have already used to achieve $$p_i$$. The optimal solution to the subproblem changes based on decisions made in the parent problem.

This is exactly the situation described in Section 15.3 regarding the longest simple path problem. There, using certain vertices in one part of the path made them unavailable for another part. Here, using certain piece lengths in one cut makes them unavailable (or reduces their availability) for subsequent cuts. In both cases, the subproblems share resources, violating the independence condition that optimal substructure requires.

{% capture note %}
The limited rod-cutting problem can still be solved, but it requires a more complex state representation. Instead of just tracking the remaining rod length, we would need to track the remaining limits for each piece length. This expands the subproblem space significantly, but dynamic programming can still be applied with this enriched state.
