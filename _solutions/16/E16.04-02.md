> Given an $$m \times n$$ matrix $$T$$ over some field (such as the reals), show that $$(S, \mathcal{I})$$ is a matroid, where $$S$$ is the set of columns of $$T$$ and $$A \in \mathcal{I}$$ if and only if the columns in $$A$$ are linearly independent.

This is the **matric matroid** (also called the **column matroid** or **linear matroid**), the structure that originally inspired Whitney to define matroids. The term "matroid" itself comes from "matrix." The idea is natural: in linear algebra, we often want to find a maximal set of linearly independent vectors, and this captures exactly that structure.

To prove $$(S, \mathcal{I})$$ is a matroid, we verify the three defining properties.

**Property 1: $$S$$ is a finite set.**

The matrix $$T$$ has $$n$$ columns, so $$S$$ is a finite set of $$n$$ vectors.

**Property 2: Hereditary property.**

We need to show that if a set of columns $$B$$ is linearly independent and $$A \subseteq B$$, then $$A$$ is also linearly independent.

This follows directly from a fundamental theorem in linear algebra: any subset of a linearly independent set is linearly independent. If the columns in $$B$$ have no nontrivial linear combination equaling zero, then certainly any subset of these columns has no such combination either.

Therefore, the hereditary property holds.

**Property 3: Exchange property.**

We need to show that if $$A$$ and $$B$$ are both sets of linearly independent columns with $$\lvert A \rvert < \lvert B \rvert$$, then there exists some column $$x \in B - A$$ such that $$A \cup \{x\}$$ is linearly independent.

This is the key property, and it follows from the **Steinitz Exchange Lemma** in linear algebra. Here is the argument:

Let $$\text{span}(A)$$ denote the vector space spanned by the columns in $$A$$. Since $$A$$ is linearly independent and has $$\lvert A \rvert$$ columns, we have $$\dim(\text{span}(A)) = \lvert A \rvert$$.

Now consider the columns in $$B$$. Since $$B$$ is linearly independent with $$\lvert B \rvert > \lvert A \rvert$$ columns, we have $$\dim(\text{span}(B)) = \lvert B \rvert > \lvert A \rvert$$.

If every column in $$B$$ were in $$\text{span}(A)$$, then $$\text{span}(B) \subseteq \text{span}(A)$$, which would imply:

$$\lvert B \rvert = \dim(\text{span}(B)) \le \dim(\text{span}(A)) = \lvert A \rvert$$

This contradicts $$\lvert B \rvert > \lvert A \rvert$$. Therefore, there must exist some column $$x \in B$$ that is not in $$\text{span}(A)$$.

Since $$x \notin \text{span}(A)$$, the column $$x$$ cannot be written as a linear combination of the columns in $$A$$. This means $$A \cup \{x\}$$ is linearly independent.

Finally, if $$x \in A$$, then $$x \in \text{span}(A)$$, contradicting our choice. So $$x \in B - A$$.

Since all three matroid properties are satisfied, $$(S, \mathcal{I})$$ is a matroid.

{% capture note %}
The rank of the matric matroid equals the rank of the matrix $$T$$ (the dimension of its column space). The maximal independent sets are exactly the bases of the column space. This connection between matroids and linear algebra runs deep: many matroid concepts like "rank," "basis," and "span" are direct generalizations of their linear algebra counterparts.
