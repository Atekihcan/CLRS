> Describe the subproblem graph for matrix-chain multiplication with an input chain of length $$n$$. How many vertices does it have? How many edges does it have, and which edges are they?

The subproblem graph captures how subproblems depend on each other. In matrix-chain multiplication, each subproblem corresponds to finding the optimal parenthesization for a contiguous subchain $$A_i A_{i+1} \cdots A_j$$ where $$1 \leq i \leq j \leq n$$.

Think of it like a dependency chart in project management: before you can solve the "full chain" problem, you need solutions to smaller chain problems, which in turn depend on even smaller chains.

**Vertices**

Each vertex represents a subproblem $$m[i,j]$$, the minimum cost to compute $$A_i \cdots A_j$$.

The number of vertices equals the number of valid $$(i,j)$$ pairs where $$1 \leq i \leq j \leq n$$:

$$\binom{n}{2} + n = \frac{n(n-1)}{2} + n = \frac{n(n+1)}{2} = \Theta(n^2)$$

The $$\binom{n}{2}$$ counts pairs with $$i < j$$ (non-trivial subproblems), and the extra $$n$$ counts the diagonal entries where $$i = j$$ (trivial single-matrix subproblems with cost 0).

**Edges**

An edge from vertex $$m[i,j]$$ to vertex $$m[i',j']$$ exists if computing $$m[i,j]$$ directly references $$m[i',j']$$.

For a subproblem $$m[i,j]$$ with $$i < j$$, we try all split points $$k$$ where $$i \leq k < j$$. For each $$k$$, we reference:
- $$m[i,k]$$: the left subchain
- $$m[k+1,j]$$: the right subchain

The number of outgoing edges from $$m[i,j]$$ is:
- 0 if $$i = j$$ (base case, no dependencies)
- $$2(j - i)$$ if $$i < j$$ (two references for each of $$j - i$$ possible split points)

The total number of edges is:

$$\begin{align*}
\sum_{i=1}^{n} \sum_{j=i+1}^{n} 2(j - i) &= 2 \sum_{i=1}^{n} \sum_{j=i+1}^{n} (j - i) \\
&= 2 \sum_{i=1}^{n} \sum_{l=1}^{n-i} l \quad \text{(substituting } l = j - i\text{)} \\
&= 2 \sum_{i=1}^{n} \frac{(n-i)(n-i+1)}{2} \\
&= \sum_{i=1}^{n} (n-i)(n-i+1) \\
&= \sum_{m=0}^{n-1} m(m+1) \quad \text{(substituting } m = n - i\text{)} \\
&= \sum_{m=1}^{n-1} (m^2 + m) \\
&= \frac{(n-1)n(2n-1)}{6} + \frac{(n-1)n}{2} \\
&= \frac{(n-1)n(2n-1) + 3(n-1)n}{6} \\
&= \frac{(n-1)n(2n+2)}{6} \\
&= \frac{(n-1)n(n+1)}{3} \\
&= \Theta(n^3)
\end{align*}$$
