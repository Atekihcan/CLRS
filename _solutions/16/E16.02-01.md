> Prove that the fractional knapsack problem has the greedy-choice property.

In the fractional knapsack problem, a thief can steal fractions of items. Each item $$i$$ has weight $$w_i$$ and value $$v_i$$, and the thief has a knapsack with capacity $$W$$. The greedy strategy sorts items by value-per-pound ($$v_i / w_i$$) in decreasing order, then takes as much as possible of each item starting from the highest value-per-pound. We need to prove that this greedy choice is always part of some optimal solution.

Consider item $$x$$ with the maximum value-per-pound ratio. We claim that there exists an optimal solution containing as much of item $$x$$ as possible (either all of $$x$$ or enough to fill the knapsack).

Suppose, for contradiction, that in some optimal solution $$O$$, we take an amount $$a < \min(w_x, W)$$ of item $$x$$. Since $$a < W$$, there must be space in the knapsack filled with some other item $$y$$ (with lower or equal value-per-pound). Let $$b$$ be the amount of item $$y$$ in the solution.

Now construct a new solution $$O'$$ by replacing $$\min(b, w_x - a)$$ pounds of item $$y$$ with the same weight of item $$x$$. Since item $$x$$ has the highest value-per-pound:

$$\frac{v_x}{w_x} \geq \frac{v_y}{w_y}$$

The value gained from the swap is:

$$\Delta = \min(b, w_x - a) \cdot \left( \frac{v_x}{w_x} - \frac{v_y}{w_y} \right) \geq 0$$

{% include ads.html %}

If item $$x$$ has strictly higher value-per-pound than item $$y$$, then $$\Delta > 0$$, contradicting the optimality of $$O$$. If they have equal ratios, the swap produces an equally optimal solution with more of item $$x$$.

By repeatedly applying this exchange argument, we can transform any optimal solution into one that takes as much as possible of the item with highest value-per-pound, without decreasing the total value.

Therefore, the greedy choice of always selecting the item with maximum value-per-pound is guaranteed to be part of some optimal solution, proving the greedy-choice property holds for the fractional knapsack problem.

{% capture note %}
The key difference between the fractional and 0-1 knapsack problems is that fractions allow us to "fill gaps" perfectly. In the 0-1 version, taking the highest value-per-pound item might leave awkward empty space that could have been filled more efficiently with smaller items.
