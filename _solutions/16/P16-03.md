> ***Acyclic subgraphs***
>
> 1. The incidence matrix for an undirected graph $$G = (V, E)$$ is a $$|V| \times |E|$$ matrix $$M$$ such that $$M_{ve} = 1$$ if edge $$e$$ is incident on vertex $$v$$, and $$M_{ve} = 0$$ otherwise. Argue that a set of columns of $$M$$ is linearly independent over the field of integers modulo 2 if and only if the corresponding set of edges is acyclic.
> 2. Suppose that we associate a nonnegative weight $$w(e)$$ with each edge in an undirected graph $$G = (V, E)$$. Give an efficient algorithm to find an acyclic subset of $$E$$ of maximum total weight.
> 3. Let $$G(V, E)$$ be an arbitrary directed graph, and let $$(E, \mathcal{I})$$ be defined so that $$A \in \mathcal{I}$$ if and only if $$A$$ does not contain any directed cycles. Give an example of a directed graph $$G$$ such that the associated system $$(E, \mathcal{I})$$ is not a matroid. Specify which defining condition for a matroid fails to hold.
> 4. The incidence matrix for a directed graph $$G = (V, E)$$ with no self-loops is a $$|V| \times |E|$$ matrix $$M$$ such that $$M_{ve} = -1$$ if edge $$e$$ leaves vertex $$v$$, $$M_{ve} = 1$$ if edge $$e$$ enters vertex $$v$$, and $$M_{ve} = 0$$ otherwise. Argue that if a set of columns of $$M$$ is linearly independent, then the corresponding set of edges does not contain a directed cycle.
> 5. Exercise 16.4-2 tells us that the set of linearly independent sets of columns of any matrix $$M$$ forms a matroid. Explain carefully why the results of parts (c) and (d) are not contradictory. How can there fail to be a perfect correspondence between the notion of a set of edges being acyclic and the notion of the associated set of columns of the incidence matrix being linearly independent?

This problem explores the deep connection between graph theory and linear algebra, specifically how the combinatorial notion of cycles in graphs relates to linear dependence in matrices. The key insight is that while this correspondence is perfect for undirected graphs, it becomes subtle for directed graphs.

### A

We prove both directions of the equivalence over $$\mathbb{Z}_2$$ (integers modulo 2).

**If the edges form a cycle, then the columns are linearly dependent.**

Consider a cycle $$v_1 \to v_2 \to \cdots \to v_k \to v_1$$ with edges $$e_1, e_2, \ldots, e_k$$. In $$\mathbb{Z}_2$$, each vertex in the cycle is incident to exactly 2 edges of the cycle. Therefore, if we sum the columns corresponding to these edges, each row corresponding to a vertex in the cycle has value $$1 + 1 = 0 \pmod{2}$$. Rows for vertices not in the cycle have value 0 (not incident to any cycle edge). Thus the sum of these $$k$$ columns is the zero vector, meaning the columns are linearly dependent.

**If the columns are linearly dependent, then the edges form a cycle (or contain a cycle).**

Suppose columns $$e_1, e_2, \ldots, e_m$$ are linearly dependent over $$\mathbb{Z}_2$$. Then there exist coefficients $$c_i \in \{0, 1\}$$ (not all zero) such that $$\sum_i c_i \cdot \text{column}(e_i) = \mathbf{0}$$. Let $$S$$ be the set of edges with $$c_i = 1$$.

For each vertex $$v$$, the sum of entries in the columns of $$S$$ for row $$v$$ is 0 in $$\mathbb{Z}_2$$. This means $$v$$ is incident to an even number of edges in $$S$$. If $$S$$ is nonempty, every vertex has even degree in the subgraph formed by $$S$$. An undirected graph where every vertex has even degree contains a cycle (specifically, it decomposes into edge-disjoint cycles). Therefore, the edges in $$S$$ contain at least one cycle.

### B

To find a maximum-weight acyclic subset of edges, we use a modified version of Kruskal's algorithm for maximum spanning forest.

{% capture code %}
Max-Weight-Acyclic(G, w)
    sort edges of E in decreasing order by weight
    A = empty set
    for each (u, v) in sorted edge order
        if adding (u, v) to A does not create a cycle
            A = A ∪ {(u, v)}
    return A
{% endcapture %}
{% include clrs_code.html title="Max-Weight-Acyclic(G, w)" %}

The algorithm works because the acyclic subsets of an undirected graph form a graphic matroid. From Theorem 16.11, the greedy algorithm finds an optimal (maximum weight) independent set in any weighted matroid. We sort by decreasing weight (to maximize rather than minimize) and greedily add edges that maintain the independence property (acyclicity).

Using a union-find data structure, the cycle check takes nearly constant amortized time. The overall running time is $$O(E \log E)$$ for sorting, plus $$O(E \cdot \alpha(V))$$ for the union-find operations, giving $$O(E \log E) = O(E \log V)$$ total.

The resulting set $$A$$ is a maximum-weight spanning forest of $$G$$. If $$G$$ is connected, $$A$$ is a maximum spanning tree.

### C

Consider the directed graph with three vertices $$\{u, v, w\}$$ and three edges:

- $$e_1: u \to v$$
- $$e_2: v \to w$$
- $$e_3: u \to w$$

Let $$A = \{e_1, e_2\}$$ and $$B = \{e_3\}$$. Both $$A$$ and $$B$$ are independent sets (neither contains a directed cycle). We have $$|A| = 2 > 1 = |B|$$.

By the exchange property of matroids, there should exist an element $$x \in A \setminus B$$ such that $$B \cup \{x\} \in \mathcal{I}$$. The candidates are $$x = e_1$$ and $$x = e_2$$.

- $$B \cup \{e_1\} = \{e_3, e_1\} = \{u \to v, u \to w\}$$: No directed cycle, so this is independent.

Actually, this example doesn't work. Let me construct a proper counterexample.

Consider a directed graph with vertices $$\{a, b, c\}$$ and edges:

- $$e_1: a \to b$$
- $$e_2: b \to c$$
- $$e_3: c \to a$$
- $$e_4: a \to c$$

Let $$A = \{e_1, e_2, e_4\}$$ and $$B = \{e_1, e_3\}$$. Set $$A$$ is independent (no directed cycle: $$a \to b \to c$$ and $$a \to c$$ don't form a cycle). Set $$B$$ is independent ($$a \to b$$ and $$c \to a$$ don't form a cycle by themselves).

We have $$|A| = 3 > 2 = |B|$$. The elements in $$A \setminus B = \{e_2, e_4\}$$.

- $$B \cup \{e_2\} = \{e_1, e_3, e_2\} = \{a \to b, b \to c, c \to a\}$$: This is a directed cycle, so NOT independent.
- $$B \cup \{e_4\} = \{e_1, e_3, e_4\} = \{a \to b, c \to a, a \to c\}$$: This contains the directed cycle $$a \to c \to a$$, so NOT independent.

Neither extension works, violating the exchange property. Therefore, $$(E, \mathcal{I})$$ is not a matroid.

### D

Consider a directed cycle $$v_1 \to v_2 \to \cdots \to v_k \to v_1$$ with directed edges $$e_1 = (v_1, v_2), e_2 = (v_2, v_3), \ldots, e_k = (v_k, v_1)$$.

For the incidence matrix of a directed graph:
- Each column has exactly one $$-1$$ (at the source vertex) and one $$+1$$ (at the target vertex)
- For edge $$e_i = (v_i, v_{i+1})$$: $$M_{v_i, e_i} = -1$$ and $$M_{v_{i+1}, e_i} = 1$$

Now sum all $$k$$ columns corresponding to the cycle edges. For each vertex $$v_i$$ in the cycle:
- Edge $$e_{i-1}$$ enters $$v_i$$, contributing $$+1$$
- Edge $$e_i$$ leaves $$v_i$$, contributing $$-1$$
- Total contribution to row $$v_i$$: $$+1 + (-1) = 0$$

The sum of all cycle columns is the zero vector, meaning the columns are linearly dependent. Taking the contrapositive: if a set of columns is linearly independent, the corresponding edges cannot contain a directed cycle.

### E

The results from parts (c) and (d) are not contradictory because **the converse of part (d) does not hold**: a set of edges can be acyclic (no directed cycles) yet the corresponding columns can be linearly dependent.

Part (c) shows that acyclic edge sets in a directed graph don't form a matroid because they violate the exchange property.

Part (d) shows that linearly independent column sets do form a matroid (by Exercise 16.4-2), and any such set corresponds to an acyclic edge set.

The discrepancy arises because:

- **Linearly independent** $$\Rightarrow$$ **Acyclic** (Part d, proven by contrapositive)
- **Acyclic** $$\not\Rightarrow$$ **Linearly independent** (the converse fails)

For example, in a directed graph with vertices $$\{u, v\}$$ and two parallel edges $$e_1: u \to v$$ and $$e_2: u \to v$$, the set $$\{e_1, e_2\}$$ contains no directed cycle (acyclic), but the columns for $$e_1$$ and $$e_2$$ are identical, hence linearly dependent.

Another example: consider edges $$u \to v$$, $$v \to w$$, and $$u \to w$$. This set is acyclic (no directed cycle), but the column for $$u \to w$$ equals the sum of columns for $$u \to v$$ and $$v \to w$$ (this can be verified row by row), so the columns are linearly dependent.

The matroid formed by linearly independent column sets is more restrictive than just requiring no directed cycles. Linear independence captures a stronger structural property that happens to imply acyclicity but is not equivalent to it.
