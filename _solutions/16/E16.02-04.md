> Professor Gekko has always dreamed of inline skating across North Dakota. He plans to cross the state on highway U.S. 2, which runs from Grand Forks, on the eastern border with Minnesota, to Williston, near the western border with Montana. The professor can carry two liters of water, and he can skate $$m$$ miles before running out of water. (Because North Dakota is relatively flat, the professor does not have to worry about drinking water at a greater rate on uphill sections than on flat or downhill sections.) The professor will start in Grand Forks with two full liters of water. His official North Dakota state map shows all the places along U.S. 2 at which he can refill his water and the distances between these locations.
>
> The professor's goal is to minimize the number of water stops along his route across the state. Give an efficient method by which he can determine which water stops he should make. Prove that your strategy yields an optimal solution, and give its running time.

The greedy strategy is simple: always travel as far as possible before stopping for water. Starting from any position, skate to the farthest water stop that is still within $$m$$ miles, refill, and repeat until reaching the destination.

Let the water stops be at positions $$0 = d_0 < d_1 < d_2 < \cdots < d_n < d_{n+1} = D$$, where $$D$$ is the total distance to Williston.

{% capture code %}
Water-Stops(d[0..n+1], m)
    // d[0] = 0 (start), d[n+1] = D (destination)
    stops = empty list
    current = 0
    while current < n + 1
        // Find farthest reachable stop from current position
        farthest = current
        while farthest + 1 ≤ n + 1 and d[farthest + 1] - d[current] ≤ m
            farthest = farthest + 1
        if farthest == current
            return "No solution: gap too large"
        if farthest < n + 1
            stops.append(farthest)
        current = farthest
    return stops
{% endcapture %}

**Correctness Proof:** We prove the greedy-choice property: the greedy first choice is always part of some optimal solution.

Let $$g_1$$ be the first stop chosen by greedy (the farthest stop within $$m$$ miles of the start). Let $$O$$ be any optimal solution with first stop $$o_1$$. If $$o_1 = g_1$$, we are done.

If $$o_1 \neq g_1$$, then $$o_1 < g_1$$ (since greedy goes as far as possible). Construct solution $$O' = (O \setminus \{o_1\}) \cup \{g_1\}$$. We need to verify $$O'$$ is feasible:
- The first segment in $$O'$$ is from start to $$g_1$$, which is valid since $$d_{g_1} \leq m$$.
- For the second segment: in $$O$$, from $$o_1$$ we reach some stop $$o_2$$ with $$d_{o_2} - d_{o_1} \leq m$$. Since $$g_1 \geq o_1$$, we have $$d_{o_2} - d_{g_1} \leq d_{o_2} - d_{o_1} \leq m$$. So from $$g_1$$ we can still reach at least $$o_2$$.

Thus $$O'$$ is feasible with the same number of stops as $$O$$, proving the greedy choice leads to an optimal solution.

**Optimal Substructure:** After making the greedy first stop at $$g_1$$, the remaining problem (getting from $$g_1$$ to the destination) is a smaller instance of the same problem. An optimal solution to the original problem consists of the greedy choice plus an optimal solution to this subproblem.

**Running Time:** The algorithm examines each stop at most twice (once as a potential destination, once as a starting point), giving $$O(n)$$ time.
