> Let $$S$$ be a finite set and let $$S_1, S_2, \ldots, S_k$$ be a partition of $$S$$ into nonempty disjoint subsets. Define the structure $$(S, \mathcal{I})$$ by the condition that $$\mathcal{I} = \{A : \lvert A \cap S_i \rvert \le 1 \text{ for } i = 1, 2, \ldots, k\}$$. Show that $$(S, \mathcal{I})$$ is a matroid. That is, the set of all sets $$A$$ that contain at most one member of each subset in the partition determines the independent sets of a matroid.

This is called a **partition matroid**. The intuition is straightforward: imagine you are forming a committee where $$S$$ represents all possible candidates, and the partition groups candidates by department (or any other category). An "independent" committee is one that has at most one representative from each department. You want to select as many people as possible while maintaining this diversity constraint.

To prove $$(S, \mathcal{I})$$ is a matroid, we verify the three defining properties.

**Property 1: $$S$$ is a finite set.**

This is given directly in the problem statement.

**Property 2: Hereditary property.**

We need to show that if $$B \in \mathcal{I}$$ and $$A \subseteq B$$, then $$A \in \mathcal{I}$$.

Since $$B \in \mathcal{I}$$, we have $$\lvert B \cap S_i \rvert \le 1$$ for all $$i$$.

Because $$A \subseteq B$$, for each $$i$$ we have $$A \cap S_i \subseteq B \cap S_i$$. Therefore:

$$\lvert A \cap S_i \rvert \le \lvert B \cap S_i \rvert \le 1$$

So $$A \in \mathcal{I}$$.

The hereditary property holds because having fewer elements means having fewer (or equal) representatives from each part.

**Property 3: Exchange property.**

Let $$A, B \in \mathcal{I}$$ with $$\lvert A \rvert < \lvert B \rvert$$. We need to find $$x \in B - A$$ such that $$A \cup \{x\} \in \mathcal{I}$$.

Since the sets $$S_1, \ldots, S_k$$ partition $$S$$, each element belongs to exactly one $$S_i$$. For an independent set $$X$$, define its **representation** as the set of indices $$\{i : X \cap S_i \neq \emptyset\}$$ (the parts from which $$X$$ takes an element).

Because $$X \in \mathcal{I}$$ means $$\lvert X \cap S_i \rvert \le 1$$ for all $$i$$, we have $$\lvert X \rvert$$ equals the number of parts that $$X$$ represents.

Since $$\lvert B \rvert > \lvert A \rvert$$, set $$B$$ represents more parts than $$A$$ does. Therefore, there exists some index $$j$$ such that:
- $$B \cap S_j \neq \emptyset$$ (B has an element from $$S_j$$)
- $$A \cap S_j = \emptyset$$ (A has no element from $$S_j$$)

Pick $$x$$ to be the unique element in $$B \cap S_j$$ (unique because $$B \in \mathcal{I}$$).

Then $$x \in B$$ and $$x \notin A$$ (since $$A \cap S_j = \emptyset$$), so $$x \in B - A$$.

For $$A \cup \{x\}$$, consider any part $$S_i$$:
- If $$i \neq j$$: $$(A \cup \{x\}) \cap S_i = A \cap S_i$$ (since $$x \in S_j$$), so $$\lvert (A \cup \{x\}) \cap S_i \rvert = \lvert A \cap S_i \rvert \le 1$$.
- If $$i = j$$: $$(A \cup \{x\}) \cap S_j = \{x\}$$, so $$\lvert (A \cup \{x\}) \cap S_j \rvert = 1 \le 1$$.

Therefore $$A \cup \{x\} \in \mathcal{I}$$.

Since all three matroid properties are satisfied, $$(S, \mathcal{I})$$ is a matroid.

{% capture note %}
The rank of a partition matroid is $$k$$ (the number of parts), since a maximal independent set picks exactly one element from each part. Partition matroids generalize the uniform matroid: if $$k = 1$$ (the entire set is one part), then $$\mathcal{I}$$ contains only $$\emptyset$$ and singletons, giving us $$U_{1,n}$$. Partition matroids arise naturally in scheduling problems where you need to select at most one task from each category.
