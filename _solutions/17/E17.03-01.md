> Suppose we have a potential function $$\Phi$$ such that $$\Phi(D_i) \ge \Phi(D_0)$$ for all $$i$$, but $$\Phi(D_0) \ne 0$$. Show that there exists a potential function $$\Phi'$$ such that $$\Phi'(D_0) = 0$$, $$\Phi'(D_i) \ge 0$$ for all $$i \ge 1$$, and the amortized costs using $$\Phi'$$ are the same as the amortized costs using $$\Phi$$.

The key insight here is that the potential method only cares about *differences* in potential between consecutive states, not the absolute values of the potential. This is similar to how in physics, we can choose any reference point for gravitational potential energy without affecting our calculations of energy changes.

Simply define the new potential function $$\Phi'$$ as:

$$\Phi'(D_i) = \Phi(D_i) - \Phi(D_0)$$

This definition shifts the entire potential function down by a constant amount, namely $$\Phi(D_0)$$.

Let us verify that $$\Phi'$$ satisfies all the required properties:

**Property 1:** $$\Phi'(D_0) = 0$$

$$\Phi'(D_0) = \Phi(D_0) - \Phi(D_0) = 0$$

**Property 2:** $$\Phi'(D_i) \ge 0$$ for all $$i \ge 1$$

Since we are given that $$\Phi(D_i) \ge \Phi(D_0)$$ for all $$i$$, we have:

$$\Phi'(D_i) = \Phi(D_i) - \Phi(D_0) \ge 0$$

**Property 3:** The amortized costs using $$\Phi'$$ are the same as those using $$\Phi$$

The amortized cost of the $$i$$th operation using $$\Phi'$$ is:

$$\begin{align*}
\hat{c}'_i &= c_i + \Phi'(D_i) - \Phi'(D_{i-1}) \\
           &= c_i + (\Phi(D_i) - \Phi(D_0)) - (\Phi(D_{i-1}) - \Phi(D_0)) \\
           &= c_i + \Phi(D_i) - \Phi(D_0) - \Phi(D_{i-1}) + \Phi(D_0) \\
           &= c_i + \Phi(D_i) - \Phi(D_{i-1}) \\
           &= \hat{c}_i
\end{align*}$$

The constant $$\Phi(D_0)$$ cancels out when we compute the difference, leaving us with the same amortized cost as the original potential function. This shows that shifting a potential function by any constant preserves all the amortized costs, which is why we have the freedom to normalize $$\Phi(D_0)$$ to zero.
