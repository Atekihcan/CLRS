> Suppose you are given two sets $$A$$ and $$B$$, each containing $$n$$ positive integers. You can choose to reorder each set however you like. After reordering, let $$a_i$$ be the $$i$$th element of set $$A$$, and let $$b_i$$ be the $$i$$th element of set $$B$$. You then receive a payoff of $$\prod_{i=1}^{n} a_i^{b_i}$$. Give an algorithm that will maximize your payoff. Prove that your algorithm maximizes the payoff, and state its running time.

The payoff is $$a_1^{b_1} \cdot a_2^{b_2} \cdots a_n^{b_n}$$. To maximize this product, we want large bases paired with large exponents. If a large base gets a small exponent while a small base gets a large exponent, we waste multiplicative potential.

The greedy strategy is: **sort both $$A$$ and $$B$$ in the same order (both increasing or both decreasing) and pair corresponding elements.**

{% capture code %}
Maximize-Payoff(A, B)
    sort A in increasing order
    sort B in increasing order
    payoff = 1
    for i = 1 to n
        payoff = payoff × A[i]^B[i]
    return payoff
{% endcapture %}

**Correctness Proof:** We use an exchange argument. Consider any pairing where some $$a_i < a_j$$ but $$b_i > b_j$$ (a "misordered" pair). We show that swapping the exponents improves or maintains the payoff.

The contribution of elements $$a_i$$ and $$a_j$$ to the payoff is $$a_i^{b_i} \cdot a_j^{b_j}$$. After swapping exponents, it becomes $$a_i^{b_j} \cdot a_j^{b_i}$$.

We need to show:

$$a_i^{b_j} \cdot a_j^{b_i} \geq a_i^{b_i} \cdot a_j^{b_j}$$

Dividing both sides by $$a_i^{b_j} \cdot a_j^{b_j}$$:

$$\left(\frac{a_j}{a_i}\right)^{b_i - b_j} \geq 1$$

Since $$a_j > a_i$$, we have $$\frac{a_j}{a_i} > 1$$. Since $$b_i > b_j$$, the exponent $$b_i - b_j > 0$$. A number greater than 1 raised to a positive power is greater than 1, so the inequality holds.

Therefore, swapping any misordered pair does not decrease the payoff. Since the sorted configuration has no misordered pairs, and any other configuration can be transformed to it through swaps that only improve the payoff, the sorted configuration is optimal.

**Optimal Substructure:** After optimally pairing the largest element of $$A$$ with the largest element of $$B$$, the remaining problem (pairing the other $$n-1$$ elements) is a smaller instance of the same problem.

**Running Time:** Sorting both arrays takes $$O(n \log n)$$. Computing the payoff takes $$O(n)$$ multiplications (each $$a_i^{b_i}$$ can be computed in $$O(\log b_i)$$ time using fast exponentiation, but since we only need the pairing, not the actual numeric result, the comparison-based complexity is $$O(n \log n)$$).

{% capture note %}
This problem illustrates the rearrangement inequality: to maximize $$\sum a_i b_i$$ (or multiplicative analogs), pair large with large. For products of powers, the same principle applies because $$\log(a^b) = b \log a$$ transforms the problem into a sum.
{% endcapture %}
