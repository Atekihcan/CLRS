> Show how to transform the weight function of a weighted matroid problem, where the desired optimal solution is a minimum-weight maximal independent subset, to make it a standard weighted-matroid problem. Argue carefully that your transformation is correct.

The standard $$\textsc{Greedy}$$ algorithm finds a **maximum-weight** maximal independent subset. But sometimes we want the opposite: a **minimum-weight** maximal independent subset. The minimum spanning tree problem is exactly this situation, as we want to connect all vertices using edges of minimum total weight.

The key insight is simple: if we flip our perspective on what "good" means, we can use the same greedy algorithm. Instead of treating high-weight elements as desirable, we make them undesirable.

**The Transformation**

Given the original weight function $$w: S \to \mathbb{R}^+$$, define a new weight function $$w': S \to \mathbb{R}^+$$ as:

$$w'(x) = W - w(x)$$

where $$W$$ is any value strictly greater than $$\max_{x \in S} w(x)$$.

This transformation ensures that $$w'(x) > 0$$ for all $$x$$ (required for the standard weighted matroid framework), and it inverts the ordering: elements that were heavy under $$w$$ become light under $$w'$$, and vice versa.

**Proof of Correctness**

We need to show that maximizing $$w'(A) = \sum_{x \in A} w'(x)$$ over all maximal independent sets $$A$$ is equivalent to minimizing $$w(A) = \sum_{x \in A} w(x)$$.

By Theorem 16.6, all maximal independent sets have the same cardinality. Let this common size be $$r$$ (the rank of the matroid).

For any maximal independent set $$A$$ with $$\lvert A \rvert = r$$:

$$\begin{align*}
w'(A) &= \sum_{x \in A} w'(x) \\
      &= \sum_{x \in A} (W - w(x)) \\
      &= \sum_{x \in A} W - \sum_{x \in A} w(x) \\
      &= r \cdot W - w(A)
\end{align*}$$

Since $$r$$ and $$W$$ are constants (the same for all maximal independent sets), we have:

$$w'(A) = rW - w(A)$$

Therefore:
- Maximizing $$w'(A)$$ is equivalent to maximizing $$rW - w(A)$$
- Since $$rW$$ is constant, this is equivalent to maximizing $$-w(A)$$
- Which is equivalent to minimizing $$w(A)$$

So running $$\textsc{Greedy}(M, w')$$ returns a maximal independent set that minimizes $$w(A)$$ among all maximal independent sets.

{% capture note %}
This transformation is exactly what the textbook uses for minimum spanning trees (page 439). There, they set $$w'(e) = w_0 - w(e)$$ where $$w_0$$ exceeds all edge weights. The greedy algorithm then finds a maximum-weight spanning tree under $$w'$$, which corresponds to a minimum-weight spanning tree under the original weights. The crucial insight is that all spanning trees have the same number of edges ($$\lvert V \rvert - 1$$), making the constant term drop out.
