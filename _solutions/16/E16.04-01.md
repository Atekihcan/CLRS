> Show that $$(S, \mathcal{I}_k)$$ is a matroid, where $$S$$ is any finite set and $$\mathcal{I}_k$$ is the set of all subsets of $$S$$ of size at most $$k$$, where $$k \le \lvert S \rvert$$.

This structure is known as the **uniform matroid** of rank $$k$$, often denoted $$U_{k,n}$$ where $$n = \lvert S \rvert$$. Think of it as the simplest possible matroid: any set of at most $$k$$ elements is considered "independent," regardless of which elements you choose. It is like having a team where any group of up to $$k$$ people can work together (no compatibility constraints), but groups larger than $$k$$ are too big to be effective.

To prove this is a matroid, we need to verify all three defining properties.

**Property 1: $$S$$ is a finite set.**

This is given directly in the problem statement.

**Property 2: Hereditary property.**

We need to show that if $$B \in \mathcal{I}_k$$ and $$A \subseteq B$$, then $$A \in \mathcal{I}_k$$.

Since $$B \in \mathcal{I}_k$$, we have $$\lvert B \rvert \le k$$. Because $$A \subseteq B$$, we know $$\lvert A \rvert \le \lvert B \rvert \le k$$. Therefore $$A \in \mathcal{I}_k$$.

The hereditary property holds because subsets of small sets are even smaller.

**Property 3: Exchange property.**

We need to show that if $$A, B \in \mathcal{I}_k$$ with $$\lvert A \rvert < \lvert B \rvert$$, then there exists some element $$x \in B - A$$ such that $$A \cup \{x\} \in \mathcal{I}_k$$.

Since $$\lvert A \rvert < \lvert B \rvert$$, there must be at least one element $$x \in B - A$$ (otherwise $$B \subseteq A$$ which would imply $$\lvert B \rvert \le \lvert A \rvert$$).

Pick any such $$x \in B - A$$. Then:

$$\lvert A \cup \{x\} \rvert = \lvert A \rvert + 1 \le \lvert B \rvert \le k$$

The last inequality holds because $$B \in \mathcal{I}_k$$. Therefore $$A \cup \{x\} \in \mathcal{I}_k$$.

Since all three matroid properties are satisfied, $$(S, \mathcal{I}_k)$$ is indeed a matroid.

{% capture note %}
The uniform matroid $$U_{k,n}$$ has a particularly nice structure. Its maximal independent sets are exactly all subsets of size $$k$$ (assuming $$k \le n$$). When $$k = n$$, every subset is independent, giving us the **free matroid**. When $$k = 1$$, only singletons and the empty set are independent. The graphic matroid from the textbook has more interesting structure because the independence condition (being acyclic) creates nontrivial dependencies between elements.
