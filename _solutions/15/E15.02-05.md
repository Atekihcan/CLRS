> Let $$R(i, j)$$ be the number of times that table entry $$m[i, j]$$ is referenced while computing other table entries in a call of $$\textsc{Matrix-Chain-Order}$$. Show that the total number of references for the entire table is
>
> $$\sum_{i=1}^{n} \sum_{j=i}^{n} R(i, j) = \frac{n^3 - n}{3}$$
>
> (Hint: You may find equation (A.3) useful.)

When we compute $$m[i,j]$$, we examine all possible split points $$k$$ from $$i$$ to $$j-1$$. Each split references $$m[i,k]$$ and $$m[k+1,j]$$. So the question becomes: for a fixed entry $$m[i,j]$$, how many times do other entries reference it?

Consider when $$m[i,j]$$ gets referenced as the "left" part of a split. This happens when we compute $$m[i,l]$$ for any $$l > j$$, using split point $$k = j$$. There are $$n - j$$ such values of $$l$$ (from $$j+1$$ to $$n$$).

Similarly, $$m[i,j]$$ gets referenced as the "right" part when we compute $$m[l,j]$$ for any $$l < i$$, using split point $$k = i - 1$$. There are $$i - 1$$ such values of $$l$$ (from 1 to $$i-1$$).

Therefore:

$$R(i,j) = (n - j) + (i - 1) = n + i - j - 1$$

Now we sum over all valid $$(i,j)$$ pairs:

$$\begin{align*}
\sum_{i=1}^{n} \sum_{j=i}^{n} R(i, j) &= \sum_{i=1}^{n} \sum_{j=i}^{n} (n + i - j - 1) \\
&= \sum_{i=1}^{n} \sum_{j=i}^{n} (n + i - 1) - \sum_{i=1}^{n} \sum_{j=i}^{n} j
\end{align*}$$

For the first sum, the inner term $$(n + i - 1)$$ does not depend on $$j$$:

$$\begin{align*}
\sum_{i=1}^{n} \sum_{j=i}^{n} (n + i - 1) &= \sum_{i=1}^{n} (n - i + 1)(n + i - 1) \\
&= \sum_{i=1}^{n} (n^2 - 1 + ni - i - n + 1 + i - i^2) \\
&= \sum_{i=1}^{n} (n^2 - n + (n-1)i - i^2) \\
&= n(n^2 - n) + (n-1)\frac{n(n+1)}{2} - \frac{n(n+1)(2n+1)}{6}
\end{align*}$$

For the second sum:

$$\begin{align*}
\sum_{i=1}^{n} \sum_{j=i}^{n} j &= \sum_{i=1}^{n} \left( \frac{n(n+1)}{2} - \frac{(i-1)i}{2} \right) \\
&= \frac{n^2(n+1)}{2} - \frac{1}{2} \sum_{i=1}^{n} (i^2 - i) \\
&= \frac{n^2(n+1)}{2} - \frac{1}{2} \left( \frac{n(n+1)(2n+1)}{6} - \frac{n(n+1)}{2} \right)
\end{align*}$$

After simplification (using equation (A.3) which states $$\sum_{k=1}^{n} k^2 = \frac{n(n+1)(2n+1)}{6}$$):

$$\sum_{i=1}^{n} \sum_{j=i}^{n} R(i, j) = \frac{n^3 - n}{3}$$

This can be verified by factoring: $$\frac{n^3 - n}{3} = \frac{n(n^2 - 1)}{3} = \frac{n(n-1)(n+1)}{3}$$.

{% capture note %}
This result confirms that $$\textsc{Matrix-Chain-Order}$$ has $$\Theta(n^3)$$ running time. The triple nested loops perform $$\Theta(n^3)$$ table references, which dominates the algorithm's complexity.
{% endcapture %}
