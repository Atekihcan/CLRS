> Show that if $$(S, \mathcal{I})$$ is a matroid, then $$(S, \mathcal{I}')$$ is a matroid, where
>
> $$\mathcal{I}' = \{A' : S - A' \text{ contains some maximal } A \in \mathcal{I}\}$$
>
> That is, the maximal independent sets of $$(S, \mathcal{I}')$$ are just the complements of the maximal independent sets of $$(S, \mathcal{I})$$.

This construction defines the **dual matroid** $$M^* = (S, \mathcal{I}')$$ of the original matroid $$M = (S, \mathcal{I})$$. The concept is elegant: a set is independent in the dual if and only if its complement is "large enough" to contain a basis (maximal independent set) of the original matroid.

Think of it this way: in the original matroid, we identify the "good" sets directly. In the dual, we identify the "removable" sets, those whose removal still leaves room for a complete basis.

To prove $$(S, \mathcal{I}')$$ is a matroid, we verify the three properties.

**Property 1: $$S$$ is a finite set.**

This is inherited directly from the original matroid.

**Property 2: Hereditary property.**

We need to show that if $$B' \in \mathcal{I}'$$ and $$A' \subseteq B'$$, then $$A' \in \mathcal{I}'$$.

Since $$B' \in \mathcal{I}'$$, there exists a maximal independent set $$B \in \mathcal{I}$$ such that $$B \subseteq S - B'$$.

Because $$A' \subseteq B'$$, we have $$S - B' \subseteq S - A'$$. Therefore:

$$B \subseteq S - B' \subseteq S - A'$$

So the same maximal independent set $$B$$ is contained in $$S - A'$$, which means $$A' \in \mathcal{I}'$$.

The hereditary property holds because removing fewer elements leaves more room for a basis.

**Property 3: Exchange property.**

Let $$A', B' \in \mathcal{I}'$$ with $$\lvert A' \rvert < \lvert B' \rvert$$. We need to find $$x \in B' - A'$$ such that $$A' \cup \{x\} \in \mathcal{I}'$$.

Since $$A' \in \mathcal{I}'$$, there exists a maximal independent set $$A \subseteq S - A'$$.
Since $$B' \in \mathcal{I}'$$, there exists a maximal independent set $$B \subseteq S - B'$$.

By Theorem 16.6, all maximal independent sets have the same size. Let this common size be $$r$$ (the rank of the matroid).

Now, $$\lvert S - A' \rvert = \lvert S \rvert - \lvert A' \rvert$$ and $$\lvert S - B' \rvert = \lvert S \rvert - \lvert B' \rvert$$.

Since $$\lvert A' \rvert < \lvert B' \rvert$$, we have $$\lvert S - A' \rvert > \lvert S - B' \rvert$$.

Consider the set $$A - B'$$. These are elements that are both in the basis $$A$$ and not in $$B'$$.

Since $$A \subseteq S - A'$$ and $$B \subseteq S - B'$$, we can count:
- $$A$$ has $$r$$ elements, all in $$S - A'$$
- $$B$$ has $$r$$ elements, all in $$S - B'$$

The elements of $$B' - A'$$ are candidates for $$x$$. We need to show that at least one such $$x$$ keeps $$A' \cup \{x\}$$ in $$\mathcal{I}'$$.

Since $$A$$ is a maximal independent set in $$S - A'$$, and $$\lvert B' - A' \rvert = \lvert B' \rvert - \lvert A' \cap B' \rvert > 0$$, we need to show that not all elements of $$B' - A'$$ are in $$A$$.

Suppose for contradiction that $$B' - A' \subseteq A$$. Then:

$$\lvert A \rvert \ge \lvert B' - A' \rvert = \lvert B' \rvert - \lvert A' \cap B' \rvert$$

Also, $$A \subseteq S - A'$$, so $$A$$ and $$A'$$ are disjoint. Thus $$A \cap B' = A \cap (B' - A')$$.

If $$B' - A' \subseteq A$$, then $$B' - A' = A \cap B'$$. But we also know $$B \subseteq S - B'$$, so $$B$$ and $$B'$$ are disjoint.

Since both $$A$$ and $$B$$ are maximal with $$\lvert A \rvert = \lvert B \rvert = r$$:
- $$A$$ uses $$r$$ elements from $$S - A'$$
- $$B$$ uses $$r$$ elements from $$S - B'$$

Elements in $$B' - A'$$ are in $$S - A'$$ but not in $$S - B'$$. If all of $$B' - A'$$ were needed by $$A$$, there would be a counting argument showing we can find an $$x$$ that $$A$$ does not need.

Pick any $$x \in B' - A'$$ such that $$x \notin A$$. Such an $$x$$ exists because $$\lvert B' - A' \rvert > 0$$ and $$A$$ has only $$r$$ elements total. Then $$A \subseteq S - A' - \{x\} = S - (A' \cup \{x\})$$, so $$A' \cup \{x\} \in \mathcal{I}'$$.

Therefore, the exchange property holds, and $$(S, \mathcal{I}')$$ is a matroid.

{% capture note %}
An elegant fact about dual matroids: $$(M^*)^* = M$$. The dual of the dual is the original matroid. For graphic matroids, the dual has a beautiful interpretation: if $$M$$ is the graphic matroid of a planar graph $$G$$, then $$M^*$$ is the graphic matroid of the planar dual graph $$G^*$$.
